Το προγραμμα

Στόχος μας είναι να επιταχύνουμε την καριέρα σου και να σε κάνουμε επαγγελματία developer, ώστε να είσαι περιζήτητος στην αγορά εργασίας ή να ξεκινήσεις την δικιά σου startup

# Δομη

Το πρόγραμμα είναι χωρισμένο σε 3 κομμάτια: Τα Θεμελιώδη , Ακαδημία, Επαγγελματική Προώθηση.

Τα Θεμελιωδη

Το πρώτο κομμάτι του προγράμματος είναι μια εντατικό εισαγωγικό σειρά μαθημάτων σχεδιασμένη για να εντρυφήσεις τις βασικές έννοιες του προγραμματισμού.

Η θεμελιώδης εκπαίδευση δεν είναι μια λίστα με links, θα δουλεύεις πάνω σ’ ένα πρόγραμμα με live καθοδήγηση και projects. Με στόχο να είσαι έτοιμος όταν έρθεις στο campus.

Ακαδημια

Τα απαραίτητα μαθηματα θα είναι από 9:00πμ μεχρι τις 6:00μμ και στην συνέχεια όποιος θέλει συνεχίζει.

Την περισσότερη ώρα θα την ξοδεύουμε σε πραγματικά projects και ασκήσεις σ’ ένα περιβάλλον που θα δουλεύετε με συμμαθητές που είναι το ίδιο αφοσιωμένοι και που θα γουστάρουν αυτό που κάνουν..

ΔΙΑΣΥΝΔΕΣΗ ΜΕ ΤΗΝ ΑΓΟΡΑ

Στο τέλος του δευτέρου μέρους θα δούμε τι σας αρέσει και θα αρχίσουμε να φτιάχνουμε το προφίλ σας, θα κάνουμε συνεντεύξεις και θα σας προετοιμάσουμε για το επόμενο σας βήμα. Όλα αυτά πριν ακόμα ξεκινήσετε τις συνεντεύξεις που θα σας έχουμε εξασφαλίσει.

ΘΕΜΕΛΙΩΔΗ

Θα σου δώσουμε ότι χρειάζεται για μπορέσεις να έχεις μια σημαντική καριέρα στο χώρο, αλλά φυσικά αυτό εξαρτάται και από εσένα, στο πόσο θα συνεχίζεις να αναπτύσεσαι προσωπικά.

Web Development

Θα μάθεις από την αρχή ως το τέλος software development χρησιμοποιώντας Node.js/Express, JavaScript, Angular.js, HTML, και CSS. Θα αναπτύξεις τρόπο σκέψης ώστε να καταλαβαίνεις προβλήματα και να βρίσκεις εφάνταστους τρόπους να τα λύσεις.

ΧΡΗΣΗ ΤΩΝ ΚΑΛΥΤΕΡΩΝ ΕΡΓΑΛΕΙΩΝ

Ο επαγγελματίας developer είναι πολύ περισσότερα από κάποιος που γραφεί κώδικα. Θα μάθεις πώς να χρησιμοποιείς τα κατάλληλα εργαλεία από source control σε web inspector και analytics.

ΚΟΥΛΤΟΥΡΑ

Θα καταλάβεις την κουλτούρα του software development που όλος ο κόσμος μιλά -τουλάχιστον στο εξωτερικό-. Θα μάθεις να δουλεύεις σε open source projects, να πηγαίνεις σε tech meetups, να κερδίζεις hackathons και να κτίσεις δίκτυο στο χώρο.

"ΝΑ ΜΠΟΡΟΥΝ ΝΑ ΜΑΘΟΥΝ "

ΑΤΟΜΑ ΠΟΥ ΝΑ ΘΕΛΟΥΜΕ ΝΑ ΔΟΥΛΕΨΟΥΜΕ ΜΑΖΙ ΤΟΥΣ

Θέλουμε να μπορούμε να περνάμε χρόνο με άτομα που να καταλαβαινόμαστε, να είναι πρόθυμα να βοηθήσουν και γενικά ευχάριστα να δουλεύεις μαζί τους.


ΥΠΟΣΤΗΡΙΚΤΕΣ


Η ΟΜΑΔΑ

Είμαστε developers απο Θεσσαλονίκη και Αθήνα που πιστεύουμε ότι χρειάζεται να καθοδηγήσουμε και να ενθαρρύνουμε το νέο κόσμο να μπει στο χώρο του software development.

Ξέρουμε οτι υπάρχει ζήτηση και θεωρούμε παράλογο ο κόσμος να μην ασχολείτε να μάθει αυτό που ζητάει η αγορά.

Αυτό που ακούσαμε είναι οτι αν κάποιος θέλει, μπορεί να ασχοληθεί. Εμείς λέμε οτι απλώς χρειάζεται να του δείξουμε τον δρόμο και τον τρόπο.


Το οικονομικο
Μακαρι να μπορουσαμε τα το κανουμε δωρεαν, αλλα πρεπει να βγαζουμε τουλαχιστον τα εξοδα

Σκοπός μας δεν είναι να βγάλουμε κέρδος, αλλά να ενώσουμε την έλλειψη στελεχών με την προσφορά μυαλών. Όμως, δεν σκοπεύουμε να χάνουμε λεφτά. Έχουμε υπολογίσει ένα τίμημα που πιστεύουμε οτι θα καλύπτει τα έξοδα μας και αυτό είναι 3000€ - 4000€

Μην ξεχνάτε οτι μιλάμε για 3*20*9 = 540 ώρες (3500/540 = 6.5€ η ωρα). Προς σύγκριση αντίστοιχα προγράμματα στην Αμερική με πολυάριθμα τμήματα, παίρνουν από $12.000 μεχρι $17.000 και στην Αγγλία από £9.600 (e.g. Hack Reactor , App Academy, Maker Academy e.t.c.)

Επιπρόσθετα, θα προσπαθήσουμε να βρούμε κάποιες επιχορηγήσεις, ώστε να μπορέσουμε να δώσουμε 1000€ υποτροφία σε κάθε σπουδαστή

Aποδεικνυουμε οτι πιστευουμε αυτά που λεμε, εμπρακτα

Θα προσπαθήσουμε να βρούμε τρόπους που να αποδεικνύουν οτι πιστεύουμε σε ότι λέμε. Δηλαδή:

1. Αν δεν έχετε τουλάχιστον μια προσφορά για δουλειά 2 μήνες μετά το πέρας του προγράμματος, θα πρέπει να σας επιστρέψουμε το μισό τίμημα.

2. Σας δίνουμε τη δυνατότητα να πληρώσετε μόνο ένα μέρος στην αρχή και όταν βρείτε δουλειά, να παίρνουμε ένα ποσό κάθε μήνα για ένα χρόνο.

Αν θεωρεις οτι το εχεις, δηλωσε ενδιαφερον
Δεν είμαστε ακόμα έτοιμοι. Βοήθησε να αποδείξουμε στους υποστηρικτές μας ότι ο υπάρχει κόσμος που θέλει να μάθει και είναι έτοιμος να βοηθήσει τον εαυτό του.

Αν σε ενδιαφέρει κάνε αίτηση και όταν είμαστε έτοιμοι για το επόμενο βήμα θα σε ενημερώσουμε.
