# Δομη

Το πρόγραμμα είναι χωρισμένο σε 3 κομμάτια: Τα Θεμελιώδη , Ακαδημία, Επαγγελματική Προώθηση.

Τα Θεμελιωδη

Το πρώτο κομμάτι του προγράμματος είναι ένα εντατικό εισαγωγικό σειρά μαθημάτων σχεδιασμένο για να εντρυφήσεις τα βασικές έννοιες του προγραμματισμού.

Η θεμελιώδης εκπαίδευση δεν είναι μια λίστα με links, θα δουλεύεις πάνω σ’ ένα πρόγραμμα, live καθοδήγηση και projects.Με στόχο να είσαι έτοιμος όταν έρθεις στο campus.

Ακαδημια

Τα απαραιτητα μαθηματα θα είναι από πρωι μεχρι το απογευμα και στην συνεχεια για οποιος θελει συνεχειζει. Σαν να πηγαινεις να μαθεις κινεζικα μετακομιζοντας στην Κινα.

Τη περισσοτερη ωρα θα την ξοδευουμε σε πραγματικα projects και ασκησεις σ’ ένα περιβαλλον που θα δουλευετε με συμαθητες που είναι το ιδιο αφοσιομενη και που θα γουσταρουν αυτό που κανουν..

ΔΙΑΣΥΝΔΕΣΗ ΜΕ ΤΗΝ ΑΓΟΡΑ

Στο τέλος του δευτέρου μέρους θα δούμε τι σας αρέσει και θα αρχίσουμε να φτιάχνουμε το προφίλ σας, θα κάνουμε συνεντεύξεις και θα σας προετοιμάσουμε για το επόμενο σας βήμα. Όλα αυτά πριν ακόμα ξεκινήσετε τις συνεντεύξεις που θα σας έχουμε εξασφαλίσει.

Being a Fullstack Developer isn't about knowing everything, but about being fearless in the face of new technologies and challenges.



# ΤΙ ΘΑ ΜΑΘΩ ?

ΘΕΜΕΛΙΩΔΗ

Θα σου δώσουμε ότι χρειάζεται για μπορέσεις να έχεις μια σημαντική καριέρα στο χώρο, αλλά φυσικά αυτό εξαρτάτε και από εσένα, στο πόσο θα συνεχίζεις να αναπτύσσεσαι προσωπικά.

Web Development

Θα μάθεις από την αρχή ως το τέλος software development χρησιμοποιώντας Node.js/Express, JavaScript, Angular.js, HTML, και CSS. Θα αναπτύξεις τρόπο σκέψεις ώστε να καταλαβαίνεις προβλήματα και να βρίσκεις αφάνταστους τρόπους να τα λύσεις.

ΧΡΗΣΗ ΤΩΝ ΚΑΛΥΤΕΡΩΝ ΕΡΓΑΛΕΙΩΝ

Ο επαγγελματίας developer είναι πολύ περισσότερα από κάποιος που γραφεί κώδικα. Θα μάθεις πώς να χρησιμοποιείς τα κατάλληλα εργαλεία από source control σε web inspector και analytics.

ΚΟΥΛΤΟΥΡΑ

Θα καταλάβεις την κουλτούρα του software development που όλος ο κόσμος μίλα -τουλάχιστον- στο εξωτερικό. Θα μάθεις να δουλεύεις σε open source projects, να πηγαίνεις σε tech meetups, να κερδίζεις hackathons και να κτίσεις δίκτυο στο χώρο.

Μαθε παραπανω για το προγραμμα σπουδων.

ΠΟΙΑ ΘΑ ΕΙΝΑΙ Η ΔΙΑΔΙΚΑΣΙΑ
1. Θα αξιολογήσουμε τις αιτήσεις των υποψηφίων που ενδιαφέρονται
2. Θα διεξάγουμε καποιες συνεντευξεις στους χωρους μας ή μεσω skype
3. Θα διαλέξουμε τέσσερα με οχτώ άτομα
4. Θα κάνουμε δοκιμαστικά μαθήματα να δούμε αν όλοι είναι όσο commiteed δηλώνουν
ΤΙ ΠΕΡΙΜΕΝΟΥΜΕ ΑΠΟ ΕΣΑΣ

Ψάχνουμε για άτομα που είναι ξέρουν τι θέλουν και είναι έτοιμη να το διεκδικήσουν. Μερικά από τα χαρακτηριστικά που κοιτάμε

ΕΤΟΙΜΑ ΝΑ ΔΟΥΛΕΨΟΥΝ ΣΚΛΗΡΑ

Το ποιο βασικό είναι να είναι έτοιμα να δουλέψουν, να ακούσουν να μάθουν. Δεν θα είναι εύκολο, αλλά κανείς δε πέτυχε τίποτα κάνοντας μόνο εύκολα…

ΝΑ ΘΕΛΟΥΝ ΠΡΑΓΜΑΤΙΚΑ ΝΑ ΜΑΘΟΥΝ

Να έχουν δείξει ότι πραγματικά θέλουν να μάθουν, τους αρέσει ο χώρος και ακόμα καλύτερα έχουν προσπαθήσει από μόνοι τους.

"ΝΑ ΜΠΟΡΟΥΝ ΝΑ ΜΑΘΟΥΝ "

Να έχουν τον τρόπο σκέψης που χρειάζεται. Δηλαδή αναλυτικός να μπορεί να σπάει τα προβλήματα σε υποπροβλήματα και να καταλαβαίνει μαθηματικά.

ΑΤΟΜΑ ΠΟΥ ΝΑ ΘΕΛΟΥΜΕ ΝΑ ΔΟΥΛΕΨΟΥΜΕ ΜΑΖΙ ΤΟΥΣ

Θέλουμε να μπορούμε να περνάμε χρόνο με άτομα που να καταλαβαινόμαστε είναι πρόθυμα να βοηθήσουν και γενικά ευχάριστα να δουλεύεις μαζί τους.


ΥΠΟΣΤΗΡΙΚΤΕΣ


Η ΟΜΑΔΑ

Ειμαστε developers απο Θεσσαλονικη και Αθηνα που πιστευουμε χρειαζεται να καθοδηγησουμε και να ενθαρρυνουμε τον νεο κοσμο να μπει στο χωρο του software development.

Ξέρουμε οτι υπάρχει ζήτηση και θεωρούμε παράλογο ο κόσμος να μην ασχολείτε να μάθει αυτό που ζητάει η αγορά.

Αυτό που ακούσαμε είναι οτι αν κάποιος θέλει, μπορεί να ασχοληθεί. Εμεις λέμε οτι απλός χρειάζεται να του δείξουμε τον δρόμο και τον τρόπο.


Το οικονομικο
Μακαρι να μπορουσαμε τα το κανουμε δωρεαν, αλλα πρεπει να βγαζουμε τουλαχιστον τα εξοδα

Σκοπος μας δεν ειναι να βγαλουμε κερδος, αλλα να ενωσουμε το ελλειψη στελεχων με την προσφορα μυαλων. Ομως δεν σκοπευουμε να χανουμε λεφτα. Εχουμε βγαλει ενα τιμημα που πιστευουμε οτι θα καλυπτει τα εξοδα μας και αυτο ειναι στις 3000€ - 4000€

Μην ξεχνατε οτι μιλαμε για 3*20*9 = 540 ωρες (3500/540 = 6.5€ η ωρα). Προς συγκριση αντιστοιχα προγραμματα στην Αμερικη με πολυαριθμα τμηματα, παιρνουν απο $12.000 μεχρι 17.000 και στην Αγγλια απο £9.600 (e.g. Hack Ractor , App Academy, Maker Academy e.t.c.)

Επιπροσθετα, θα προσπαθησουμε να παρουμε καποια επιχορηγηση, ωστε να μπορουμε να δινουμε 1000€ υποτροφια

Aποδεικνυουμε οτι πιστευουμε αυτα που λεμε, εμπρακτα

Θα προσπαθησουμε να βρουμε τροπους που να αποδεικνιουν οτι πιστευουμε σ' οτι λεμε. Δηλαδη:

1. Αν δεν εχετε τουλαχιστον μια προσφορα για δουλεια 2 μηνες μετα το περας του προγραμματος, θα πρεπει να σας επιστρεψουμε το μισο τιμημα

2. Σας δινουμε τη δυνατοτητα να πληρωσετε μονο ενα μερος στην αρχη και οταν βρειτε δουλεια, να παιρνουμε ενα ποσο καθε μηνα για ενα χρονο.

Αν θεωρεις οτι το εχεις, δηλωσε ενδιαφερων
Δεν ειμαστε ακομα ετοιμοι. Βοηθησε να αποδειξουμε στους υποστηρικτές μας οτι ο υπάρχει κοσμος που θελει να μαθει και ειναι ετοιμος να βοηθήσει τον εαυτό του.

Αν δε ενδιαφερει κανε αιτηση και οταν ειμαστε ετοιμοι για το επομενο βημα θα σε ενημερωσουμε.
