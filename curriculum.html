The Προγραμμα Σπουδων

Cutting-edge Knowledge. Deep Understanding.

Fullstack's curriculum will expose you to the latest in modern software development for the Internet. We will train you to not only understand the latest technologies, but also instill a passion for discovery and learning that will keep you at the top of your game through the generations of new technologies that emerge over time.

Our weapon of choice to teach software engineering is fullstack JavaScript - the most popular and widely supported language in the world. The future of the web is dynamic and real-time JavaScript applications and you will graduate from Fullstack an expert on hacking with JS.

While graduating an expert JavaScript hacker will make you invaluable in the job market, the real transformation you will experience at Fullstack is a feeling of empowerment where the barrier will no longer be the ability to build the software but rather thinking of the problems you want to solve.

# Computer Science

We are standing today on the shoulders of giants: the pioneering computer scientists of the past century who built the bedrock of computing and networking that underlies everything we use. Being a great developer means not only being fluent with the latest frameworks and languages, but also having an evolved intuition for writing efficient and beautiful code. This is why your journey at Fullstack will start with a deep dive into the fundamentals of Computer Science.

Data Structures

Data Structures are fundamental to CS as they allow developers to work with complex data, while balancing processing power and memory requirements. At Fullstack you will learn the inner wizardry of commonly used data structures like arrays, hash tables, lists, trees and heaps by not only using them, but also writing them yourself.

Algorithms

Understanding algorithms is an integral part of being a developer. At Fullstack, you will not only learn to write the whole gamut of essential CS algorithms like sorting, searching, encryption, but you will also know which one to choose at the right time.

Big-O Space/Time Analysis

The spirit of efficiency in CS stems from the early days and continues today. At Fullstack, you will learn to evaluate and optimize an algorithm for it's efficiency in terms of both space and time using asymptotic time analysis.

# Programming Mastery

Being a great developer starts with being a great programmer. In Foundations and at the Academy, we will drill in core programming concepts to ensure your fluency in speaking and writing the language. Once there, then you're ready to move beyond the language to focus on systems and frameworks and ultimately full-stack applications.

Object-Oriented Programming

OO programming is a popular approach to designing modular, reusable software systems. We will learn many flavors of OO programming as used in JavaScript including how it is applied in today's software for the Internet.

Asynchronous Programming

Async programming is one of the most exciting features of programming in JS and Node.js, and possibly also one of the most misunderstood. You will learn to untangle callbacks and be a master of using the wildly popular async library.

Recursion

Recursion is an advanced method of solving problems based on the divide and conquer strategy. It is widely tested in interviews and commonly used to solve complicated algorithms with lucid code. At Fullstack, we'll dive deep into recursion by writing code and using fun in-class demonstrations and exercises to truly understand its power.

Regular Expressions

Regular expressions are a powerful and (mostly) standardized way of searching, replacing, and parsing text with complex patterns of characters. Regexes are indispensable to programming and we will explore their power in detail at Fullstack.

# Frontend Applications

Angular.js/JavaScript MVC

JavaScript is redefining what it means to be a modern web application. We'll start with using jQuery for simple page changes and move on to study design patterns of "Single Page Web Applications" using Angular.js. You will know why/where/how JavaScript frameworks are being used to build advanced web client applications like Gmail, Facebook and Google Maps.

HTML5

HTML is the markup language that started it all and is still the backbone of web information. HTML5 adds new contextual tags, semantics and new capabilities to web browsers. While HTML is one of the easier languages to master, it's a critical part of the stack.

CSS3

If HTML is the architect, CSS is the interior designer. Know how CSS decorates HTML and go beyond to know how to create simple but effective designs for your projects. Use tools like SCSS along with Bootstrap to make your sites beautiful and responsive.

# Backend Applications

Node.js/Express

JavaScript is an incredibly flexible and popular language and quickly becoming the "lingua franca" of the web. Node.js and JavaScript based back-end development is becoming the leader in next generation backend web engineering. You will be writing and thinking JavaScript from Day 1 and be an expert by graduation.

Ruby on Rails

The framework that launched a thousand startups, Ruby on Rails is one of the most widely deployed frameworks and lauded for it's flexibility, extensibility and library support. You will learn Ruby and discover how it is used increasingly today, as an API layer to a MVC JS front-end.

Services and Apis

APIs allow us to connect to vast amounts of data and functionality. For example, we will learn about systems of authentication via OAuth and building payment systems using APIs by providers like Stripe and Balanced.

# Data

MongoDB/NoSQL

Beyond traditional relational databases, there are a wide variety of "NoSQL" datastores that service different needs. We'll learn how to think about where and why to use these new technologies.

SQL

SQL is the language that powers the communication lines of our web applications by sending and fetching data about customers, purchases and traffic from the database. We'll learn to write SQL (and then also how to avoid writing it using Rails).

Relational Databases (Postgres)

Postgres has become one of the most powerful (free) data management tools available. We'll become knowledgeable on Postgres's various capabilities and how to use powerful new features of Postgres in our applications.

# Tools & Best Practices

Git Version Control

If you've ever worked on a project where someone named a file "FINAL - Proposal - DY.doc", you know why Version Control is necessary. Go beyond just version control, learn how to use Git to effectively drive your workflow.

Text Editors

Vim? Emacs? Sublime Text? We won't start a flame war here, but you'll be a text wrangling expert before the semester is over. Whatever editor you choose, we can show you the hidden corners where the experts hide their productivity secrets.

Test-Driven Development

Testing code with code is a fundamental aspect of scalability and reliability. Learn where and why to test-drive your development.

Shell & Systems

If a monitor is a computer's face, then a Shell prompt is a window into its soul. Learn how to be comfortable in the Shell (Terminal, Console) and how to control machines using Shell Scripting.
