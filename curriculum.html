The Προγραμμα Σπουδων

ΓΝΩΣΗ ΤΩΝ ΑΠΑΙΤΗΣΕΩΝ ΤΗΣ ΑΓΟΡΑΣ ΚΑΙ ΤΗΣ ΚΑΤΕΥΘΥΝΣΗ ΤΗΣ ΤΕΧΝΟΛΟΓΙΑΣ.

Το πρόγραμμα σπουδών θα σας βασίζεται στη τελευταία trends του web development. Θα σας εκπαιδεύσουμε όχι μόνο να καταλαβαίνετε τις τελευταίες τεχνολογίες αλλά να θέλετε να ξανακαλύπτετε ότι καινούριο συμβαίνει και να το μαθαίνετε μόνοι σας ώστε να είστε γνώστες στο που πηγαίνει η αγορά.

Επιλέγουμε να δουλεύουμε καταρχάς με JavaScript καθώς είναι η πιο διάσημη και χρησιμοποιούμενη γλώσσα προγραμματισμού στον κόσμο. Το μέλλον στο web είναι δυναμικό , real-time και με την JavaScript θα μπορείτε να ανταποκριθείτε. Στη συνέχεια θα σας παρουσιάσουμε την Ruby on Rails που είναι ένα διαδεδομένο και εύκολο framework και χρησιμοποιείτε από πολλές μικρότερες εταιρίες.

# Computer Science

Το να είσαι καλός developer δεν σημαίνει μόνο να ξέρεις τα τελευταία frameworks και γλώσσες, αλλά το να γράφεις αποτελεσματικό και καλό κώδικα. Γι’αυτό το λόγο ξεκινάμε με ένα καλό overview στα θεμελιώδη της πληροφορικής.

Data Structures

Τα Data Structures είναι θεμελιώδης γνώση, καθώς επιτρέπουν τους developers να δουλεύουν με πολύπλοκα δεδομένα, ενώ ταυτόχρονα μειώνουν την χρήση CPU και Memory. Εδώ θα μάθουμε τα πιο σημαντικά data structures, όπως arrays, hash tables, lists, trees και heaps όχι μόνο με το να τα χρησιμοποιούμε αλλά με το να τα γράφουμε μόνοι μας.

Algorithms

Το να καταλαβαίνει κάποιος αλγορίθμους είναι βασικό μέρος του να είσαι developer. Εδώ θα μάθουμε να γράφουμε όλους τους απαραίτητους αλγορίθμους όπως searching, encryption, αλλά και ποιον να χρησιμοποιήσετε την κάθε φορά.

Big-O Space/Time Analysis

Θα μάθουμε να αξιολογούμε και να βελτιστοποιούμε έναν αλγόριθμο για την απόδοση από την άποψη χώρου και χρόνου, χρησιμοποιώντας asymptotic time analysis.

# Programming Mastery

Ένας πολύ καλός developer ξεκινά με το να είναι πολύ καλός στο κώδικα. Θα δουλέψουμε πολύ στο να γίνετε πολύ δυνατή στον κώδικα και μετά θα προχωρήσουμε για να επικεντρωθεί σε frameworks και τελικά σε full-stack applications.

Object-Oriented Programming

OO programming είναι δημοφιλης στο σχεδιασμο, τμηματικων, επαναχρησιμοποιησιμων δομων. Θα μαθετε ΟΟ σε javascipt και πως δουλευετε σημερα στο internet.

Asynchronous Programming

Async programming είναι ένα από τα πιο συναρπαστικά χαρακτηριστικά του προγραμματισμού σε JS and Node.js, και, ενδεχομένως, επίσης, ένα από τα πιο παρεξηγημένα. Θα μάθετε να δουλεύετε callbacks και την χρήση του δημοφιλών async βιβλιοθηκών.

Recursion

Recursion είναι μια προηγμένη μέθοδος για την επίλυση προβλημάτων με βάση τη στρατηγική διαίρει και βασίλευε. Είναι ευρέως δοκιμαστεί σε συνεντεύξεις και συνήθως χρησιμοποιείται για την επίλυση περίπλοκων αλγορίθμων.

Regular Expressions

Regular expressions είναι ένα ισχυρό και (κυρίως) τυποποιημένο τρόπο για την αναζήτηση, την αντικατάσταση και την ανάλυση κειμένου με πολύπλοκα σχήματα χαρακτήρων. Regexes είναι απαραίτητη για τον προγραμματισμό και θα δούμε τη δυναμική τους λεπτομερώς.

# Frontend Applications

Angular.js/JavaScript MVC

Η JavaScript επαναπροσδιορίζει το τι σημαίνει μια σύγχρονη web εφαρμογή. Θα ξεκινήσουμε με τη χρήση jQuery για απλές αλλαγές σελίδα και να προχωρήσουμε για να μελετήσει τα σχεδιαστικά πρότυπα του "Single Page Web Applications" με Angular.js. Θα ξέρετε γιατί / πού / πώς JavaScript πλαίσια που χρησιμοποιούνται για την κατασκευή προηγμένων εφαρμογών web client, όπως το Gmail, το Facebook και το Google Maps.

HTML5

Η HTML είναι markup γλώσσα που τα ξεκίνησε όλα και παραμένει η ραχοκοκαλιά του web. Η HTML5 προσθέτει νέες ετικέτες, τη και νέες δυνατότητες στο web browsers. Ενώ η HTML είναι ένα από τα ευκολότερα γλώσσες, είναι ωστόσο και πολύ κρίσιμη.

CSS3

Εάν η HTML είναι ο αρχιτέκτονας, η CSS είναι ο διακοσμητής. Γνωρίστε πώς η CSS ομορφαίνει μια σελίδα HTML και πώς να δημιουργείτε μια απλό αλλά αποτελεσματικό σχεδιασμό για τα projects σας. Χρησιμοποιήστε εργαλεία όπως το SCSS μαζί με Bootstrap να καθιστούν τις ιστοσελίδες σας όμορφες και responsive.

# Backend Applications

Node.js/Express

JavaScript είναι μια απίστευτα ευέλικτη και δημοφιλής γλώσσα και να γίνει γρήγορα το «lingua franca» του web. Node.js (JavaScript back-end) έχει γίνει ο ηγέτης στην επόμενη γενιά backend web development. Θα γράφεις και σκέφτεσαι JavaScript από την 1 ημέρα 1 και στο τέλος θα είσαι ένας JS guru.

Ruby on Rails

Το framework που ξεκίνησε χιλιάδες startups, Ruby on Rails ένα από τα πιο ευρέως διαδεδομένα frameworks λόγο της ευελιξίας, επεκτασιμότητας και η υποστήριξης στις βιβλιοθήκες Θα μάθετε Ruby και ανακαλύψτε πώς χρησιμοποιείται όλο και περισσότερο σήμερα, από ένα API έως ένα MVC JS front-end.

Services and Apis

Ta APIs μας επιτρέπουν να συνδέουμε ένα τεράστια μεγέθη δεδομένων με μεγάλη λειτουργικότητα. Για παράδειγμα, θα μάθουμε για τα συστήματα ελέγχου ταυτότητας μέσω OAuth και τα συστήματα πληρωμών κτίριο με χρήση APIs από παρόχους όπως Stripe και Balanced.

# Data

MongoDB/NoSQL

Πέρα από τις παραδοσιακές βάσεις δεδομένων, υπάρχει μια μεγάλη ποικιλία από "NoSQL" datastores που εξυπηρετεί διαφορετικές ανάγκες. Θα μάθετε πώς να σκέφτεστε για το πού και γιατί να χρησιμοποιούν αυτές τις νέες τεχνολογίες.

SQL

Η SQL είναι η γλώσσα που κυρίως χρησιμοποιείτε στην επικοινωνία των εφαρμογών web, για την αποστολή και ανάκτηση δεδομένων σχετικά με τους πελάτες, τις αγορές και την κίνηση τους από την βάση δεδομένων. Θα μάθουν να γράφουν SQL (και στη συνέχεια, επίσης, πώς να αποφύγετε να γράψετε, χρησιμοποιώντας Rails).

Relational Databases (Postgres)

Η Postgres έχει γίνει ένα από τα πιο ισχυρά εργαλεία (δωρεάν) διαχείρισης δεδομένων. Θα γίνετε γνώστες των διάφορων δυνατοτήτων της Postgres και πώς να χρησιμοποιούν αυτά τα νέα ισχυρά χαρακτηριστικά στις εφαρμογές μας.

# Tools & Best Practices

Git Version Control

Εάν έχετε εργαστεί ποτέ σε ένα project όπου κάποιος ονομάζει ένα αρχείο "ΤΕΛΙΚΟ - Πρόταση - Ημερομηνια.doc", ξέρετε γιατί Version Control είναι απαραίτητο. Πηγαίνετε πέρα από τα όρια του ελέγχου της έκδοσης, θα μάθουμε πώς να χρησιμοποιούμε το Git για να ελέγχετε την αποτελεσματική ροή των projects σας.

Text Editors

Vim? Emacs? Sublime Text? Υπάρχουν πολλοί επεξεργαστές κώδικα. Όποιον θα επιλέξετε, εμείς μπορούμε να σας παρουσιάσουμε τα μυστικά της παραγωγικότητάς τους.

Test-Driven Development

Να τεστάρεις κώδικα με κώδικα είναι ένα θεμελιώδες στοιχείο της κλιμάκωσης αξιόπιστα την εφαρμογή σου.

Shell & Systems

Εάν η οθόνη είναι το πρόσωπο ενός υπολογιστή, το Shell prompt είναι ένα παράθυρο στην ψυχή του. Μάθετε πώς να είστε άνετοι με το Shell (Terminal, Console) και πώς να ελέγχετε μηχανές που χρησιμοποιούν Shell Scripting.
